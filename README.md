# Bytevora Solutions

**From idea to launch — software built for real business impact.**

Modern company website for Bytevora Solutions, a software startup offering custom web development, mobile apps, SaaS platforms, AI integration, and digital transformation services.

## Tech Stack

- **React** + **Vite** — Fast development and optimized builds
- **Tailwind CSS v4** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Lucide React** — Clean icon set
- **React Hook Form** — Form handling and validation

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run preview
```

Optimized for deployment on **Vercel**.

## Structure

```
src/
  components/   — Reusable components (Navbar, Footer, etc.)
  sections/     — Page sections (Hero, Services, Portfolio, etc.)
  hooks/        — Custom React hooks
  utils/        — Utility functions
  assets/       — Static assets
```
